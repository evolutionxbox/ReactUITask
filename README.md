# ReactUITask
An approach to completing a Senior Dev UI Task using React.

## Getting started

### Install

```bash
cd ReactUITask
npm install
```

or

```bash
cd ReactUITask
yarn
```

### Running

```bash
npm start
```

Open http://localhost:9009 in your favourite web browser.
